# AI Interview Platform

**Безопасная интеллектуальная платформа для автоматизированного технического собеседования IT-специалистов с использованием ИИ и встроенного VPN-сервиса для шифрования и защиты данных.**

Дипломная работа © 2025

---

## Технологический стек

| Компонент | Технология | Версия |
|-----------|-----------|--------|
| Фронтенд | React + Vite + TailwindCSS + Zustand + Recharts | 18.x / 5.x / 3.x |
| Бэкенд | FastAPI (Python) + SQLAlchemy async | 0.104 / 2.0 |
| База данных | SQLite (разработка) / PostgreSQL (продакшен) | 3.x / 15.x |
| Десктоп | Electron | 28.x |
| AI/LLM | OpenRouter API (LLaMA 3.3 70B) — бесплатный | - |
| Авторизация | JWT (HS256) + bcrypt | - |
| Отчёты | ReportLab (PDF) + openpyxl (Excel) | 4.0 / 3.1 |
| VPN | WireGuard (архитектура) | 1.x |

## Быстрый старт

### Первая установка (один раз)
```powershell
cd frontend
npm install
cd ..\electron
npm install
cd ..\backend
pip install -r requirements.txt
```

### Запуск (каждый раз)
```powershell
.\start.bat
```
Скрипт автоматически: собирает фронтенд → копирует в Electron → запускает бэкенд → открывает приложение.

## Безопасность

| Механизм | Реализация |
|----------|-----------|
| Аутентификация | JWT токены (HS256, TTL 8 часов) |
| Пароли | bcrypt хэширование (cost factor 12) |
| Роли | RBAC — 4 уровня (Admin/HR/Candidate/Guest) |
| Античит | Детекция вставки кода, переключения вкладок, DevTools |
| Аудит | Журнал всех действий с IP-адресами и временными метками |
| VPN | WireGuard туннель (ChaCha20-Poly1305) — архитектура |
| Шифрование | HTTPS/TLS для API, AES-256 для БД (план) |
